# Регистрация нового пользователя

## Функционал

Экран регистрации `RegisterUserScreen` открывается из профиля и позволяет создать аккаунт.

**Поля формы:**
- Логин, Email, Пароль
- Имя, Пол, Дата рождения (по умолчанию сегодняшняя)
- Страна и Город (через nested sheet)
- Принятие пользовательского соглашения

**Валидация:**
- Пароль: минимум 6 символов
- Email: корректный формат
- Дата рождения: не в будущем (возраст не проверяется)

**Подсказки:**
- Логин: латиница, цифры, подчёркивание (без валидации)

**Навигация:**
- Выбор страны/города через отдельные экраны в sheet
- Sheet нельзя закрыть свайпом
- После выбора города автоматически подставляется страна

**API и данные:**
- Регистрация через `SWApi.register()` с `@FormUrlEncoded`
- Возвращает модель `User`, которая сохраняется в базу
- Токен сохраняется, профиль загружается без лишнего запроса

**Обработка ошибок:**
- Ошибки отображаются в Snackbar внутри sheet
