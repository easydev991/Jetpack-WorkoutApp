**Задача:** Реализовать экран для создания и редактирования площадок (парков) с двумя режимами работы.

## Этап 6 (TDD): Интеграция с существующими экранами ⚠

### 6.0.1 ParksRootViewModel (Infrastructure) ✅

**Обзор:** Создать ViewModel для `ParksRootScreen` чтобы упростить тестирование и устранить проблему с MockK `relaxed = true` + suspend operator `invoke()`.

**Проблема:** `mockk(relaxed = true)` некорректно работает с suspend operator function. При вызове `mockLocationHandler.invoke()` возвращается сам mock-объект вместо `Result<NewParkDraft>`, что вызывает `ClassCastException: kotlin.Result cannot be cast to NewParkDraft`.

**Решение:** Создать `IParksRootViewModel` + `ParksRootViewModel` с `StateFlow`/`SharedFlow`, использовать `FakeParksRootViewModel` в androidTest вместо MockK.

#### Архитектура

```
ParksRootScreen (Compose)
    ↓
IParksRootViewModel (interface)
    ↓
ParksRootViewModel (implementation)
    ↓
ICreateParkLocationHandler, Activity (для permission checks)
```

**Примечание:** `UserNotifier` и `ResourcesProvider` удалены из зависимостей — логика упрощена.

#### Файлы для создания

| Файл | Назначение |
|------|------------|
| `IParksRootViewModel.kt` | Интерфейс с State/Events/Actions |
| `ParksRootUiState.kt` | Data class для UI state |
| `ParksRootEvent.kt` | Sealed class для one-shot events |
| `ParksRootViewModel.kt` | Реализация с логикой |
| `FakeParksRootViewModel.kt` | Stub для androidTest |

#### IParksRootViewModel

```kotlin
interface IParksRootViewModel {
    val uiState: StateFlow<ParksRootUiState>
    val events: SharedFlow<ParksRootEvent>
    val permissionLauncher: ((Map<String, Boolean>) -> Unit)?
        set

    fun onFabClicked()
    fun onPermissionResult(permissions: Map<String, Boolean>)
    fun onDismissDialog()
    fun onConfirmDialog()
    fun onOpenSettings(intent: Intent)
}

data class ParksRootUiState(
    val showPermissionDialog: Boolean = false,
    val permissionDialogCause: PermissionDialogCause? = null
)

sealed class ParksRootEvent {
    data class NavigateToCreatePark(val draft: NewParkDraft) : ParksRootEvent()
    data object OpenSettings : ParksRootEvent()
}

enum class PermissionDialogCause { DENIED, FOREVER_DENIED }
```

#### ParksRootViewModel зависимости

```kotlin
class ParksRootViewModel(
    private val createParkLocationHandler: ICreateParkLocationHandler,
    private val activity: Activity
) : ViewModel(), IParksRootViewModel
```

**Стало:**
```kotlin
@Composable
fun ParksRootScreen(
    viewModel: IParksRootViewModel = viewModel {

            activity = activity as Activity
        )
    }
)
```
