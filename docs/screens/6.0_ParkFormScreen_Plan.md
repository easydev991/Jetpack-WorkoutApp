# План: ParkFormScreen - Создание/редактирование площадки

## Обзор

**Задача:** Реализовать экран для создания и редактирования площадок (парков) с двумя режимами работы.

**Референс:** Android `EventFormScreen.kt`; iOS `ParkFormScreen.swift` + `ParkForm.swift`

**Статус:** Этапы 1-5 завершены. Stage 5.4 в основном реализован. Остались интеграционные тесты. Этапы 6.1/6.3 pending.

## Основные компоненты

### iOS ParkForm (ParkGrade/ParkSize)
| Поле | Тип | Описание |
|------|-----|----------|
| `address` | String | Адрес площадки |
| `latitude` | String | Широта |
| `longitude` | String | Долгота |
| `cityId` | Int? | ID города |
| `typeId` | Int | Тип площадки (ParkGrade) |
| `sizeId` | Int | Размер площадки (ParkSize) |
| `photosCount` | Int | Количество существующих фото |
| `newMediaFiles` | [MediaFile] | Новые фото |

### Android ParkType / ParkSize
| ParkType | rawValue | R.string |
|----------|----------|----------|
| SOVIET | 1 | `soviet_park` |
| MODERN | 2 | `modern_park` |
| COLLARS | 3 | `collars_park` |
| LEGENDARY | 6 | `legendary_park` |

| ParkSize | rawValue | R.string |
|----------|----------|----------|
| SMALL | 1 | `small_park` |
| MEDIUM | 2 | `medium_park` |
| LARGE | 3 | `large_park` |
