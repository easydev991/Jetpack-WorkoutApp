## Четвертая итерация: Исправление багов 🔴

### ✅ Баг 1: FAB-кнопка не отображается

**Проблема:** FAB-кнопка для создания новой записи не отображалась.

**Решение:** Добавлен `Box` с отступами (start, top, end), bottom padding пропущен для FAB. Удалены отладочные логи.

**Связанные файлы:** `JournalEntriesScreen.kt`, `JournalsListScreen.kt` (референс), `JournalEntriesViewModel.kt`

---

### ✅ Баг 2: Пустое поле при редактировании записи

**Проблема:** При открытии `TextEntryScreen` для редактирования записи текстовое поле пустое.

**Решение:** Добавлена функция `getInitialText()` в `TextEntryViewModel`, которая извлекает `oldEntry` из `EditInfo` для режимов редактирования.

**Связанные файлы:** `TextEntryViewModel.kt`, `TextEntryViewModelTest.kt`, `TextEntryScreenTest.kt`

---

### ✅ Баг 3: Список записей не обновляется после сохранения изменений

**Проблема:** При сохранении изменений в `TextEntryScreen` на экране со списком записей ничего не обновляется, текст записи остается прежним.

**Решение:** Добавлено обновление локального кэша в `SWRepository.editComment()` для Journal опций через `JournalEntryDao.getById()` и `insert()`. UI обновляется автоматически через Flow. При редактировании `loadEntries()` не вызывается.

**Связанные файлы:** `JournalEntryDao.kt`, `SWRepository.kt`, `AppContainer.kt`, `JournalEntriesScreen.kt`

---

### ✅ Баг 4: TextEntryScreen не блокирует вертикальные жесты свайпа

**Проблема:** `TextEntryScreen` можно потянуть вниз с самого верха, но он потом пружинит наверх. Нужно заблокировать все вертикальные жесты свайпа по аналогии с `LoginSheetHost` (есть `Modifier.disableAllGestures()`).

**Решение:** Добавлен модификатор `disableAllGestures()` к `TextEntryScreen` в `TextEntrySheetHost`. Это блокирует все жесты (вертикальные и горизонтальные) для всего контента sheet, предотвращая случайное закрытие шторки жестами.

**Детали реализации:**
1. **TextEntrySheetHost:** Добавлен `Modifier.disableAllGestures()` к `TextEntryScreen`
2. **UI-тесты:** Добавлен тест `textEntryScreen_withLongText_thenTextFieldIsScrollable()` для проверки интерактивности TextField

**Связанные файлы:**
- `TextEntrySheetHost.kt` - применение модификатора
- `LoginSheetHost.kt` - референсная реализация `disableAllGestures()`
- `disableAllGestures.kt` - реализация модификатора
- `TextEntryScreenTest.kt` - добавлен тест для проверки TextField
