# План: EventFormScreen - Создание/редактирование мероприятия

## Обзор

**Задача:** Реализовать экран для создания и редактирования мероприятий с тремя режимами работы.

**Референс:** iOS `SwiftUI-WorkoutApp/Screens/Events/EventFormScreen.swift`

**Маршруты навигации (уже существуют):**
- `Screen.CreateEvent` - создание мероприятия
- `Screen.EditEvent` - редактирование мероприятия
- `Screen.CreateEventForPark` - создание мероприятия для выбранной площадки (out of scope)

## Режимы работы экрана

### 1. regularCreate
- **Источник:** EventsScreen → FAB
- **Параметры:** нет
- **Заголовок:** "Новое мероприятие"
- **Площадка:** выбирается из списка площадок пользователя

### 2. editExisting
- **Источник:** EventDetailScreen → кнопка редактирования
- **Параметры:** eventId, существующие данные мероприятия
- **Заголовок:** "Мероприятие"
- **Площадка:** выбирается из списка площадок пользователя (если их > 1)

### 3. createForSelected (out of scope)
- **Источник:** ParkDetailScreen
- **Параметры:** parkId, parkName
- **Заголовок:** "Новое мероприятие"
- **Площадка:** предустановлена, не редактируется
